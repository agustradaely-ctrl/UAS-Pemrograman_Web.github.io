Project: Website Dinamis

Pertemuan ini merupakan puncak pembelajaran dengan membangun website dinamis secara utuh, mencakup perencanaan proyek, perancangan database, pengembangan frontend, integrasi backend, hingga persiapan deployment.

Project Planning

Tahap perencanaan mencakup penentuan tujuan website, fitur utama, target pengguna, serta pembagian tugas antara frontend dan backend.

Database Design

Database dirancang menggunakan pendekatan relasional dengan memperhatikan normalisasi dan relasi antar tabel.

Contoh Skema Tabel:
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nama VARCHAR(100),
    email VARCHAR(100),
    password VARCHAR(255)
);

Frontend Development

Frontend dikembangkan menggunakan HTML, CSS, dan JavaScript untuk membangun antarmuka yang responsif dan interaktif.

Contoh Struktur HTML:
<header>
    <h1>Website Dinamis</h1>
</header>

<main>
    <section>Konten</section>
</main>

Backend Integration

Backend bertugas mengelola data, autentikasi, serta komunikasi antara database dan frontend.

Contoh PHP Backend:
<?php
$conn = mysqli_connect("localhost", "root", "", "db_web");

$result = mysqli_query($conn, "SELECT * FROM users");
?>

Deployment Preparation

Tahap deployment mencakup konfigurasi server, keamanan aplikasi, serta optimasi performa sebelum website dipublikasikan.
