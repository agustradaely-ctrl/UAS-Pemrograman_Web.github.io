HTML5: Struktur & Semantik

Pertemuan ini membahas konsep dasar HTML5 dengan fokus pada struktur dokumen dan elemen semantik agar website lebih terstruktur, mudah dipahami, dan aksesibel.

Doctype & Struktur Dasar

DOCTYPE digunakan untuk memberi tahu browser bahwa dokumen menggunakan standar HTML5. Struktur dasar HTML terdiri dari elemen <html>, <head>, dan <body>.

Contoh Source Code:
<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Website HTML5</title>
</head>
<body>
    Konten Website
</body>
</html>

Elemen Semantik HTML5

Elemen semantik memberikan makna pada struktur halaman sehingga membantu SEO dan aksesibilitas.

Contoh Source Code:
<header>
    Header Website
</header>

<nav>
    Menu Navigasi
</nav>

<main>
    <article>
        Konten Utama
    </article>
</main>

<footer>
    Footer Website
</footer>

Form dan Input Types Baru

HTML5 menyediakan tipe input baru yang mendukung validasi otomatis dan meningkatkan pengalaman pengguna.

Contoh Source Code:
<form>
    <input type="email" placeholder="Email">
    <input type="date">
    <input type="number" min="0" max="100">
    <input type="color">
    <button>Kirim</button>
</form>

Audio dan Video HTML5

HTML5 memungkinkan pemutaran audio dan video tanpa plugin tambahan.

Contoh Source Code:
<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
</audio>

<video controls width="320">
    <source src="video.mp4" type="video/mp4">
</video>

Canvas Dasar

Elemen canvas digunakan untuk menggambar objek grafis menggunakan JavaScript.

Contoh Source Code:
<canvas id="kanvas" width="200" height="100"></canvas>

<script>
const canvas = document.getElementById("kanvas");
const ctx = canvas.getContext("2d");

ctx.fillStyle = "blue";
ctx.fillRect(20, 20, 160, 60);
</script>
