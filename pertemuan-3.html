CSS3: Styling Modern

Pertemuan ini membahas konsep styling modern menggunakan CSS3 untuk membangun tampilan website yang terstruktur, fleksibel, dan interaktif.

Selector dan Specificity

Selector digunakan untuk memilih elemen HTML yang akan diberi gaya. Ketika beberapa aturan CSS diterapkan pada elemen yang sama, browser akan menentukan prioritas berdasarkan specificity.

Urutan prioritas secara umum adalah: Inline > ID > Class > Elemen. Pemahaman specificity penting untuk menghindari konflik gaya.

Box Model & Positioning

Box Model menjelaskan bagaimana elemen HTML ditampilkan sebagai kotak yang terdiri dari content, padding, border, dan margin. Konsep ini penting untuk mengatur jarak dan ukuran elemen.

Selain itu, properti position digunakan untuk menentukan bagaimana elemen diposisikan relatif terhadap dokumen atau elemen lain.

Contoh Source Code:


.box {
    width: 200px;
    padding: 20px;
    border: 2px solid black;
    margin: 15px;
}

.box-relative {
    position: relative;
    top: 10px;
    left: 20px;
}

Flexbox Layout

Flexbox adalah sistem layout satu dimensi yang digunakan untuk menyusun elemen dalam baris atau kolom secara fleksibel. Flexbox sangat cocok untuk navigasi, card layout, dan komponen UI.

Contoh Source Code:


.flex-container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

CSS Grid

CSS Grid merupakan sistem layout dua dimensi yang memungkinkan pengaturan baris dan kolom secara bersamaan. Grid sangat efektif untuk layout halaman yang kompleks.

Transition & Animation

CSS menyediakan fitur transition untuk membuat perubahan gaya secara halus, serta animation untuk membuat animasi berbasis keyframes tanpa JavaScript.

Contoh Source Code:


.button {
    transition: background-color 0.3s ease;
}

@keyframes fadeIn {
    from {
        opacity: 0;
    }
    to {
        opacity: 1;
    }
}
