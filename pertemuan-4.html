JavaScript Dasar

Pertemuan ini membahas fundamental pemrograman JavaScript yang meliputi variabel, tipe data, operator, serta kontrol alur program sebagai dasar untuk membangun logika aplikasi web.

Variabel (let, const, var)

Variabel digunakan untuk menyimpan data di dalam program. JavaScript menyediakan tiga cara mendeklarasikan variabel, yaitu var, let, dan const. Penggunaan let dan const direkomendasikan karena memiliki scope yang lebih aman dibandingkan var.

Contoh Source Code:

var nama = "Budi";
let umur = 21;
const statusMahasiswa = true;

Tipe Data dan Operator

JavaScript memiliki beberapa tipe data dasar seperti string, number, boolean, dan undefined. Operator digunakan untuk melakukan operasi matematika, perbandingan, maupun logika.

Contoh Source Code:

let a = 10;
let b = 5;

let jumlah = a + b;
let lebihBesar = a > b;

Kondisional (if, switch)

Struktur kondisional digunakan untuk menjalankan kode berdasarkan kondisi tertentu. if digunakan untuk keputusan sederhana, sedangkan switch cocok untuk banyak pilihan kondisi.

Contoh Source Code:

let nilai = 80;

if (nilai >= 75) {
    console.log("Lulus");
} else {
    console.log("Tidak Lulus");
}

let hari = "Senin";

switch (hari) {
    case "Senin":
        console.log("Hari Kerja");
        break;
    case "Minggu":
        console.log("Hari Libur");
        break;
}

Perulangan (for, while)

Perulangan digunakan untuk menjalankan kode secara berulang selama kondisi tertentu terpenuhi. Perulangan membantu menghindari penulisan kode yang berulang-ulang.

Contoh Source Code:

for (let i = 1; i <= 5; i++) {
    console.log(i);
}

let x = 1;
while (x <= 3) {
    console.log(x);
    x++;
}

Function dan Scope

Function adalah blok kode yang dapat digunakan kembali. Scope menentukan di mana variabel dapat diakses, baik di dalam function maupun di luar function.

Contoh Source Code:

function hitungLuas(panjang, lebar) {
    let luas = panjang * lebar;
    return luas;
}

let hasil = hitungLuas(5, 4);
