DOM Manipulation

Pertemuan ini membahas cara memahami dan memanipulasi Document Object Model (DOM) menggunakan JavaScript untuk mengubah struktur, konten, dan perilaku halaman web secara dinamis.

Konsep DOM Tree

DOM Tree merepresentasikan dokumen HTML sebagai struktur hierarki berbentuk pohon. Setiap elemen HTML merupakan sebuah node yang memiliki hubungan parent, child, dan sibling.

Selektor DOM (querySelector)

Metode querySelector digunakan untuk memilih elemen HTML menggunakan sintaks selektor CSS sehingga mudah dan fleksibel.

Contoh Source Code:

const heading = document.querySelector("h1");
const tombol = document.querySelector(".btn");

Mengubah Konten & Style

JavaScript dapat mengubah konten teks, HTML, serta gaya visual elemen secara langsung melalui DOM.

Contoh Source Code:

const paragraf = document.querySelector("p");

paragraf.textContent = "Konten telah diperbarui";
paragraf.style.color = "teal";
paragraf.style.fontWeight = "bold";

Event Handling

Event handling memungkinkan JavaScript merespons interaksi pengguna seperti klik, hover, atau input.

Contoh Source Code:

const button = document.querySelector("button");

button.addEventListener("click", function () {
    console.log("Tombol ditekan");
});

Membuat Elemen Dinamis

Elemen HTML dapat dibuat dan ditambahkan ke dalam DOM secara dinamis menggunakan JavaScript tanpa menulis langsung di HTML.

Contoh Source Code:

const elemenBaru = document.createElement("p");
elemenBaru.textContent = "Elemen ini dibuat secara dinamis";

document.body.appendChild(elemenBaru);
