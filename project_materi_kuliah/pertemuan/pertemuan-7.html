Pertemuan 7

Bootstrap Framework

Pertemuan ini membahas penggunaan framework Bootstrap untuk mempercepat pengembangan website melalui sistem grid, komponen siap pakai, utilities, serta kustomisasi tampilan.

Grid System Bootstrap

Grid System Bootstrap menggunakan sistem 12 kolom untuk membangun layout responsif.

Contoh Source Code:
<div class="row">
    <div class="col-md-6">Kolom 1</div>
    <div class="col-md-6">Kolom 2</div>
</div>

Komponen Bootstrap

Bootstrap menyediakan komponen UI siap pakai seperti tombol dan card.

Contoh Source Code:
<button class="btn btn-primary">
    Tombol Bootstrap
</button>

Utilities dan Helpers

Utilities adalah class kecil untuk pengaturan cepat seperti margin, padding, dan alignment.

Contoh Source Code:
<div class="text-center mt-4 p-3 bg-light">
    Konten dengan utility Bootstrap
</div>

Customizing Bootstrap

Bootstrap dapat dikustomisasi dengan menimpa class bawaan menggunakan CSS.

Contoh Source Code (CSS):
.btn-primary {
    background-color: #4f46e5;
    border: none;
}

Bootstrap Icons

Bootstrap Icons adalah kumpulan ikon SVG resmi dari Bootstrap.

Contoh Source Code:
<i class="bi bi-bootstrap-fill"></i>
