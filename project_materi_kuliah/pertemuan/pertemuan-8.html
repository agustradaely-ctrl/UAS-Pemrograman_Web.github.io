PHP Dasar

Pengenalan pemrograman server-side menggunakan PHP untuk memproses data dan menghasilkan halaman web dinamis.

Sintaks Dasar PHP

PHP adalah bahasa pemrograman server-side yang dieksekusi di server dan hasilnya dikirim ke browser dalam bentuk HTML. Kode PHP ditulis di dalam tag <?php ... ?>.

Source Code:
<?php
echo "Hello, World!";
?>

Variable dan Tipe Data

Variabel dalam PHP diawali dengan tanda $. PHP bersifat dynamically typed, artinya tipe data ditentukan saat runtime.

Source Code:
<?php
$nama = "Agustra Kenaditor Daeli";
$umur = 22;
$isMahasiswa = true;
?>

Control Structures

Control structure digunakan untuk mengatur alur program, seperti pengambilan keputusan menggunakan if atau switch.

Source Code:
<?php
$nilai = 85;

if ($nilai >= 75) {
    echo "Lulus";
} else {
    echo "Tidak Lulus";
}
?>

Array dan Looping

Array digunakan untuk menyimpan banyak data dalam satu variabel, sedangkan looping digunakan untuk mengulang proses.

Source Code:
<?php
$buah = ["Apel", "Jeruk", "Mangga"];

foreach ($buah as $item) {
    echo $item;
}
?>

Function PHP

Function digunakan untuk mengelompokkan kode agar dapat digunakan kembali dan membuat program lebih terstruktur.

Source Code:
<?php
function salam($nama) {
    return "Halo, " . $nama;
}

echo salam("Agustra");
?>
